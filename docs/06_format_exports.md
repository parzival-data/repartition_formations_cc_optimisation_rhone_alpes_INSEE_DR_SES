# Format des exports

Les exports cibles sont :

- un CSV des formations ouvertes ;
- un CSV des affectations commune -> formation ;
- un rapport Markdown de resolution ;
- un journal des assouplissements appliques ;
- optionnellement un fichier XLSX de synthese.

Le rapport devra indiquer la configuration utilisee, le statut du solveur, les valeurs separees des trois termes d'objectif et la composition de chaque formation.
